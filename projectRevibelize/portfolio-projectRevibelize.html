Overview:

Project Revibelize is a project designed to be placed at events. A virtual busker interacts with people who walk past, encouraging them to dance.

These individuals are tracked using an Azure Kinect. An avatar of the people passing by is displayed in-game, and depending on which plant they stand near, different layers of the music will start playing. The intensity of the music also changes based on how intensely players dance.

The project requires at least 4 people to activate and play all the music layers.

Context:

This game was created for a group project spanning an entire semester. The group consisted of 3 artists, 3 programmers, and a musician from outside of DAE.

For this game, we got to use a Microsoft Kinect from DAE Research.

I was mainly responsible for the implementation of sounds.

Tools used:

  • C#
  • Unity
  • Wwise
  • Reaper
  • Azure Kinect

Reflection:

This project taught me more about working in a team. I really enjoyed being able to try out different input methods beyond just mouse and keyboard thanks to the Azure Kinect.
