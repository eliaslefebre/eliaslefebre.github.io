Summary:

This project was an art project made during Game Art 2 while I was following the Indie Game Development program. The idea for the game revolved around a story of a man who falls asleep on an airplane and dreams that he died. If developed further, the game would have taken him through all his core memories.

The focus of this project was creating art assets.

My Contribution:

  • 3D modeling of props
  • 3D character modeling
  • Making environments in Unity
  • Programming in C#
